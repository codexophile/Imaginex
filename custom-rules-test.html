Debugging Tips

  • Open the browser console (F12) to see debug messages
  • Look for messages like "Element matches custom rule: [Rule Name]"
  • Check for "Custom rule generated URL: [URL]" to see the extracted URL
  • If a rule isn't working, verify the CSS selector matches the element
  • Test your JavaScript in the console first: document.querySelector('your-selector')
