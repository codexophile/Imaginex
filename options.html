Shortcuts

Assign up to two shortcuts per action. Works with keyboard or mouse inputs.

Tips: press Escape to cancel capture. Scroll to use wheel shortcuts. Mouse buttons include left, middle, right, back, and forward.

Cloud Sync

Synchronize your settings with Google Drive. Merges cloud and local settings, then saves the combined result. Sign in with your Google account once.

API Keys

Store API keys for rules that fetch high-res images from external services (YouTube, Twitter, etc.). Keys are stored locally and referenced in rules via {settings.keyName}.

Built-in Rules

These are hardcoded patterns and behaviors that help the extension detect and display images across various sites. Toggle individual rules on or off to customize the extension's behavior.

Site-Specific CSS Fixes

Custom Rules

Define custom rules to find higher-quality images for specific elements. For example, find maxresdefault thumbnails for YouTube videos, high-res product images, or return a set of URLs to browse as a gallery.
